My Projects

Real Estate Predictive Algorithm Using R

October-December 2023

  • Worked with a partner to create a predictive algorithm to determine median real estate value in California based on input values.
  • Found a dataset online via Kaggle and cleaned and prepared the data for analysis
  • Conducted a single variable linear regression model using SSE's (sum of squares error)
  • Found a high correlation between our target variable (median home value) and median income.
  • Began to develop a multivariable regression, we determined variable selection with forward selection process- starting with the variable with lowest SSE, and adding the variable that resulted in the lowest SSE when in combination with the first, and continued.
  • We added the most effective attributes to our predictive algorithm, based on the results of the multivariable regression.
  • We developed the prediction model using the predict() function in R, and found we had an SAE of about 40k.
